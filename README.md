# pixiv-spider
A simple programe to crawl the picture on the ranking of pixiv

## Require
- requests
- json
- threading
- datetime

## Usage
```
python pixiv.py
```

## Description
Add function: crawl R18 pictures (daily/weekly)
Copy cookie to cookie.txt if you need R18 mode
